 **Resumo do Sistema**
O **QuizEducaSP** é um sistema desenvolvido para testar conhecimentos do usuário por meio de perguntas de múltipla escolha.  
O objetivo é oferecer uma experiência simples, rápida e intuitiva, permitindo ao usuário revisar suas respostas.

 **Como jogar o Quiz**
1. Efetue o cadastro ou login.  
2. Escolha sua forma de estudos.  
3. Escolha a alternativa desejada.
